داشبورد API پروژه فروشگاه

احراز هویت (JWT)

دریافت توکن:

POST /token/

رفرش توکن:

POST /token/refresh/

تأیید توکن:

POST /token/verify/

محصولات

لیست محصولات:

GET /products/

افزودن محصول:

POST /products/

سبد خرید

افزودن به سبد خرید:

POST /cart/add/

صفحات معمولی (shop.urls)

خانه:

GET /api/home/

مشاهده سبد خرید:

GET /api/view_cart/

ساخت سفارش:

POST /create_order/

سفارش‌های فروشنده:

GET /api/seller_order/

ثبت نظر:

POST /api/review/

نمایش نظرات محصول:

GET /api/ProductReviewsView/

Swagger

مستندات API:

GET /api/swagger/
